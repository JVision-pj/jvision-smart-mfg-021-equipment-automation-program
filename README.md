# EAP預警塔

半導體/PCB設備與MES間之自動化整合軟體，實現設備自動控制與資料傳輸。

- 原始系統：EAP（Equipment Automation Program，設備自動化整合平台）
- 產業分類：智慧製造｜製造管理
- 使用部門：設備工程、製造工程、資訊部門
- 工作流程：MES指令下達 → EAP轉譯 → 設備執行 → 製程數據回傳 → MES記錄比對
- AI 應用：AI比對配方執行異常並預警、智慧優化設備通訊排程
- 主要 KPI：自動化涵蓋率、配方錯誤率、資料傳輸準確率
- 代表產品：Cimetrix EAP Studio、Brooks Software、Applied Materials相關整合平台
